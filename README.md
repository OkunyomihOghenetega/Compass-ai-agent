# Compass AI – Spiritual AI Assistant (Jotform AI-Based)

## Overview
Compass AI is a no-code AI-powered assistant built using Jotform AI tools. It is designed to provide structured spiritual guidance, reflection, and supportive conversational responses.

## Purpose
This project explores how AI systems can simulate meaningful, human-centered interactions using no-code AI tools.

It focuses on:
- Structured conversational AI design
- Guided response flows
- User interaction logic
- Spiritual reflection support (Bible-based guidance)

## What It Does
- Responds to spiritual and reflective questions  
- Provides structured guidance and encouragement  
- Simulates AI assistant behavior using logic and prompts  
- Demonstrates no-code AI agent design principles  

## Tools Used
- Jotform AI Agent Builder  
- No-code workflow and AI logic system  

## Important Note
This is a no-code AI prototype and not a replacement for spiritual leaders or religious guidance systems.
